## Overview

The `abb_egm_state_controller` package provides a `ros_control`-based controller, which publishes the current state of *Externally Guided Motion* (`EGM`) communication channels into the ROS system.

Please see [abb_robot_driver](https://github.com/ros-industrial/abb_robot_driver) for more details.
